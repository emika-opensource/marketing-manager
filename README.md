# Marketing Command Center

Emika AI Employee for comprehensive marketing operations.

## Features
- Brand & ICP Management
- Ad Account Management (Google, Meta, LinkedIn, TikTok)
- Influencer Marketing Pipeline
- Creative Studio (fal.ai integration)
- Channel Strategy
- Campaign Tracking & Budget Management

## Setup
```bash
npm install
node server.js
```

Server runs on port 3000.

## Environment Variables
- `FAL_KEY` — fal.ai API key for creative generation
- `DATA_DIR` — Custom data directory (default: `/home/node/emika/marketing-hub/`)
